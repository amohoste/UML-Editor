UML Viewer - Help

Visualiseren 

Aanroep 

Via commandolijn

 $ java -jar 'naam jar' [bestandslocatie] [screenshotlocatie]
  • Zonder argumenten: programma start met leeg werkblad
  • Één argument: programma start en visualiseert meegegeven xml - bestand
  • Twee argumenten: programma maakt een screenshot van het meegegeven xml-bestand en sluit opnieuw af
  • Meer argumenten: argumenten worden genegeerd en gebruiker wordt hiervan op de hoogte gebracht

Export as png

Maakt een screenshot van het werkblad en slaat deze op op de door de gebruiker aangeduide locatie. 

Klasse verwijderen

Selecteer de gewenste klasse en druk op de backspace toets.

Klassen toevoegen

  • Selecteer in het menu edit -> new box
  • Geef de naam van de klasse in en druk op ok
  • Er verschijnt nu een blauwe overlay over het scherm en de cursor verandert naar een kruisje om aan te geven dat je kan tekenen
  • Druk en schuif ergens op het scherm om de box aan te maken
